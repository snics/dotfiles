# Backlog & Roadmap

A simple to-do list and roadmap for dotfiles improvements. Work on items when time and motivation align!

## 📝 How to Write Tasks

Use this format: `- [ ] {emoji} **{Module}** - {Description} → [Details](ideas/xyz.md) · [📁 Files](path)`

- **Emoji** = Type of work (✨ feat, 🐛 fix, 📝 docs, 🔧 config, etc.)
- **Module** = Which area (asdf, tmux, zsh, git, docs, etc.)
- **Description** = Short, clear, actionable
- **Links** = Optional: → [Details](ideas/file.md) for planning, · [📁 Files](path) for code

See [planning/README.md](README.md) for full guidelines and examples.
