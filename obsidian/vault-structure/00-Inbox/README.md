# 📥 Inbox

**Purpose:** Quick capture zone for all incoming thoughts, ideas, tasks, and information.

## Guidelines

- **Capture everything here first** - Don't worry about organization
- **Process regularly** (daily or weekly)
- **Move items to appropriate folders** after processing
- **Keep it empty** - Inbox Zero approach

## Processing Workflow

1. Review each item
2. Decide: 
   - Is it actionable? → Move to Project or create task
   - Is it reference? → Move to Resources
   - Is it part of ongoing responsibility? → Move to Areas
   - Is it no longer relevant? → Move to Archive or delete

## Quick Capture

Use the **Quick Capture** template (`Cmd+Shift+T`) for fast entry.
