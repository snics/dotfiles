# 🌱 Areas

**Purpose:** Ongoing responsibilities and areas of life that require continuous attention.

## What is an Area?

An area is:
- **Responsibility-based** - Ongoing commitment
- **No end date** - Continues indefinitely
- **Standards-based** - Measured by maintenance, not completion

Examples:
- Health & Fitness
- Personal Development
- Career & Professional Growth
- Relationships
- Finances
- Home & Living

## Suggested Sub-folders

```
02-Areas/
├── Development/        # Professional growth, learning
├── Health/            # Fitness, nutrition, wellness
├── Personal/          # Personal growth, habits
├── Relationships/     # Family, friends, networking
├── Finance/           # Budget, investments, planning
├── Daily Notes/       # Daily journal entries
└── Weekly Notes/      # Weekly reviews
```

## Maintaining Areas

- Review regularly (weekly/monthly)
- Set standards and habits
- Track progress over time
- Create projects when needed for improvements

---

*Areas never "complete" - they are maintained over time.*
