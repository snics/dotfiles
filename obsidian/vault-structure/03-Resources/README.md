# 📚 Resources

**Purpose:** Reference materials, knowledge base, and information you want to save.

## What goes in Resources?

- Books summaries and notes
- Article highlights and annotations
- Course materials and learnings
- Research and documentation
- How-to guides and tutorials
- Templates and frameworks
- Code snippets and examples
- Quotes and inspiration

## Suggested Organization

```
03-Resources/
├── Books/             # Book notes and summaries
├── Articles/          # Saved articles and highlights
├── Courses/           # Course notes and materials
├── Documentation/     # Technical docs, references
├── Tools/             # Tool guides, tips & tricks
├── Code/              # Code snippets, examples
├── People/            # People to learn from
└── Topics/            # Organized by subject matter
```

## Resource Types

### Literature Notes
- Summary of source material
- Key concepts and ideas
- Your interpretations

### Permanent Notes (Zettelkasten)
- Atomic ideas in your own words
- Connected to other concepts
- Use Zettelkasten template

## Tagging Strategy

Use tags to cross-reference:
- `#book`, `#article`, `#course`
- `#programming`, `#design`, `#business`
- `#concept`, `#framework`, `#tool`
