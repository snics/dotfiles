# 🎯 Projects

**Purpose:** Active projects with specific goals and deadlines.

## What is a Project?

A project is:
- **Goal-oriented** - Has a clear end state
- **Time-bound** - Has a deadline or target completion date
- **Action-oriented** - Requires multiple steps to complete

Examples:
- Launch new feature
- Learn a new technology
- Write a blog post series
- Plan a trip

## Project Structure

Each project should have:
- Clear goal and success criteria
- List of tasks and milestones
- Timeline and deadlines
- Resources and links
- Status tracking

Use the **Project** template for new projects.

## Project Lifecycle

1. **Planning** - Define goals, scope, timeline
2. **In Progress** - Active execution
3. **Review** - Evaluate results
4. **Complete** - Move to Archive with learnings
